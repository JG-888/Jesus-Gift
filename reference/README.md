Annotated charts from JesseG for the low-float parabolic-short strategy. The agent is
multimodal, so it can load these to recognize the pattern **visually** — the mountain →
**roll-over** → drop — not just from the text in `config/rules.md`.

- **01-support-resistance-diagram.png** — what support & resistance levels are (the concept).
- **02-EQ-daily-resistance.png** — EQ (Equillium), *daily*: a prior **144M-volume** day set hard
  resistance at **1.36**; price pushed to 1.25 and failed there. Shows how a high-volume day becomes
  overhead resistance.
- **03-EQ-1min-short-entry.png** — EQ, *1-minute*: JesseG shorted **1.24** anticipating it couldn't break
  the **1.36** resistance; risk set at 1.36. The actual entry + stop placement.
- **04-AGMH-1min-open-drop.png** — AGMH, *1-minute*: a clean open drop (**9.78 → 7.70**). AGMH was a
  **watchlist** candidate ("a possible short Monday or Tuesday") — a multi-day example.
- **05-JFB-5day-multiday.jpeg** — JFB on the Yahoo **5-day** view: the multi-day spike (9/29) into the
  next-day setup (9/30). The owner's key **multi-day** example — this is the exact view used to judge how
  much inverted-parabola correction is left.
- **06-BQ-5day-example.jpeg** — BQ (Boqii) on the Yahoo 5-day view: another spike → fade example.
